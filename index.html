Discord bot project

Lespri Bot

Lespri Bot is a Discord community assistant for NO M.E.G.A.T.R.O.N. It handles role management, color roles, reaction roles, welcome/onboarding, privacy-first image generation, moderation support, and experimental narrated-chat systems.

Roles & Colors

Self-assignable roles, reaction-role panels, color roles, and role-count reports.

Privacy-first Generation

Temporary image artifacts, prompt scrubbing, short-lived cache, and no durable local job folders.

Narrated Chats

Planned quest and roleplay rooms backed by a multi-model relay architecture.

Public policy

Privacy Policy

Lespri is designed to minimize retained data. Local generated-image files are temporary work artifacts, prompts should be sanitized before provider calls, and operational logs should avoid raw prompts, image bytes, API tokens, and unnecessary personal information.

Core doctrine: collect only what is needed, retain as little as possible, and make user-facing controls clear.
