# Lespri Bot

Lespri Bot is a Discord community assistant for NO M.E.G.A.T.R.O.N. It handles role management, color roles, reaction roles, welcome/onboarding flows, moderation support, privacy-first image generation, and experimental narrated-chat systems.

<h2>Current functions</h2>

- Self-assignable roles and color roles.
- Reaction-role panels and role-count summaries.
- Welcome/onboarding guidance.
- Privacy-first image generation workflow.
- Short-lived temporary caches instead of durable local image archives.
- Moderator and owner maintenance commands.
- Planned narrated chat and quest/roleplay systems.

<h2>Privacy-first design</h2>

Lespri is designed to minimize retained data. Local generated-image files are temporary work artifacts, prompts should be sanitized before provider calls, and administrative logs should avoid raw prompt or image retention.
