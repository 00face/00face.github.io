Lespri Bot

Discord automation, role systems, privacy-first generation, moderation support, and narrated AI architecture.

Overview

Lespri Bot is the Discord automation and ritual interface for NO M.E.G.A.T.R.O.N. It supports role systems, onboarding, reaction roles, moderation tools, privacy-first image generation, and future narrated AI rooms.

Doctrine: Lespri should help the server run cleanly while collecting as little durable data as possible.

Feature Matrix

Roles

Self-assign roles, moderator role actions, governance checks, and public role commands.

Color Roles

One active color role per user, with automatic clearing of older hues.

Reaction Roles

Emoji-driven assignment from a dedicated roles-and-colors panel.

Welcome Flow

Onboarding buttons for Anime, NSFW, Game Night, and Movie Night access.

Image Generation

Prompt parsing, reroll, delete, download, local upscale, and delivery status.

Privacy Tools

Privacy checks, purge flows, short-lived cache, and user forget controls.

Privacy Architecture

Prompt Minimization

Scrub obvious personal data before provider calls where practical.

Temporary Files

Generated image artifacts are work products, not permanent archives.

Short Cache

Reroll, upscale, and download actions should use short-lived memory.

Metadata Logs

Prefer status, action type, and IDs over raw prompts or image bytes.

Governance

Lespri follows server hierarchy and role boundaries. Quarantined status denies sensitive access first. Moderators arbitrate. Members receive normal amenities. Color roles are cosmetic and carry no authority.

Owner

Final server authority.

Moderator

Dispute arbitration and safety support.

Members

Community access and support.

Color Roles

Identity and aesthetics only.
